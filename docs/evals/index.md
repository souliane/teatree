# Weekly behavioral-eval dashboard

This page is the published dashboard for the weekly metered behavioral-eval run.

It is regenerated by the `publish` job of `.github/workflows/eval.yml`: after the
metered suite fans out across `{lane, shard}` legs, that job merges every shard's
*sanitized* summary (counts + total cost + a `scenario | lane | verdict | trials | cost`
table — never a transcript) into this file via `scripts/eval/merge_summaries.py`, commits it to
`main`, and the push triggers `docs.yml` to serve it on GitHub Pages.

Until the first weekly run lands, this placeholder is what is served. The private
per-trial transcripts (the agent's reasoning and tool calls) live in the run's
`eval-report-*` artifacts, never on this public page.
